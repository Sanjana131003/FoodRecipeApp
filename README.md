
## GitHub Project Description
A modern Full Stack Food Recipe Web Application built using React.js, Node.js, Express.js, and MongoDB. The platform allows users to create, edit, delete, favorite, and share recipes with a beautiful responsive UI and secure authentication system. Users can explore recipes from other users, manage their own recipes, and maintain personalized favorite collections.
✨ Features:
* User Authentication (Login / Signup)
* Add, Edit, Delete Recipes
* Favorite & Share Recipes
* View Recipes from Other Users
* Personalized My Recipes Section
* Responsive Modern UI
* MongoDB Database Integration
* JWT Authentication
* Dark Mode Support
* Search & Filter Recipes
Tech Stack:
React.js | Tailwind CSS | Node.js | Express.js | MongoDB | JWT | Axios

# 🍽️ Food Recipe Web Application
A modern and responsive Full Stack Food Recipe Web Application where users can discover, create, manage, and share delicious recipes.
## 🚀 Features
### 🔐 Authentication
* User Signup & Login
* JWT-based Authentication
* Secure Password Hashing using bcrypt

### 🍲 Recipe Management
* Add New Recipes
* Edit Existing Recipes
* Delete Recipes
* Upload Recipe Images
* View Detailed Recipe Information

### ❤️ Favorites System
* Add Recipes to Favorites
* Remove Recipes from Favorites
* Personalized Favorites Page

### 👤 User-Specific Functionality
* My Recipes page shows only logged-in user's recipes
* Home page displays recipes from all users
* Favorites section displays only liked recipes

### 🌟 Extra Features
* Search Recipes
* Filter by Category
* Responsive Mobile-Friendly UI
* Toast Notifications
* Smooth Animations
* Dark Mode Support

# 🛠️ Tech Stack

## Backend

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/food-recipe-app.git
cd food-recipe-app
```

---

## 2️⃣ Setup Backend
```bash
cd backend
npm install
```
Create a `.env` file inside backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
Run Backend:

```bash
npm start


## 3️⃣ Setup Frontend

```bash
cd frontend
npm install
npm run dev
